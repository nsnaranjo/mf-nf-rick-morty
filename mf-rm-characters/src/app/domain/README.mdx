# Capa del Dominio

El dominio es la capa definida por el **negocio**, es decir, no debe estar atada a 
ningún concepto técnico y dentro de ella vemos las entidades y los casos de uso.

## Entidades

Clases que representan conceptos del dominio con sus atributos y métodos.

## Interfaces

Contratos que definen cómo interactuar con las capas externas (por ejemplo, repositorios o 
servicios).

## Lógica de Negocio

Regla de negocio que no depende de frameworks ni de tecnologías externas.
