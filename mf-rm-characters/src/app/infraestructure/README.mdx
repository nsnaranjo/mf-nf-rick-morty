# Capa de Infraestructura

La capa de infraestructura se refiere a los componentes del sistema que permiten la comunicación 
con servicios externos, bases de datos, frameworks, bibliotecas, y cualquier otra dependencia 
externa que el sistema necesite para funcionar. Esta capa se encarga de la implementación 
técnica de las interfaces y protocolos definidos por las capas de dominio y aplicación.

## Repositorios

Implementaciones concretas para acceder a la base de datos.

## Adaptadores

Clases que conectan servicios externos con la aplicación

## Servicios Externos

Conexión a APIs, autenticación, etc.

## Helpers (helpers)

Es una capa dedicada a ayudar a sus hermanos (miembros de la capa de infraestructura) con 
transformaciones, operaciones, funciones útiles para la capa de infraestructura.
