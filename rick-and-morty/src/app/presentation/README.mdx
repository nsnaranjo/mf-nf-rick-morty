# Capa de Presentación

Esta capa contiene la interfaz de usuario y la lógica asociada. Es la capa que interactúa 
directamente con el usuario y muestra datos procesados por la aplicación.

## Components

Implementa la interfaz de usuario, incluyendo componentes visuales y su lógica asociada. 
Aquí es donde se presentan los datos al usuario y se manejan las interacciones del usuario 
con la aplicación.
