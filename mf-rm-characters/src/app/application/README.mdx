# Capa de Aplicación

Orquesta el flujo de datos hacia y desde la capa de dominio, implementando casos de uso o 
servicios de aplicación. No contiene lógica de negocio, solo coordinación.

## Casos de Uso (Use Cases)

Servicios que coordinan la lógica de negocio para cumplir con un caso de uso específico.

## DTOs (Data Transfer Objects)

Objetos simples que transportan datos entre la capa de aplicación y otras capas.
